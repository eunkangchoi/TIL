함수

  • 함수 또한 자료형이다.
  • 함수는 인자 /매개변수/ 리턴값을 갖는다.
  • 함수 선언
    • 선언적 함수: function 키워드에 함수명을 작성하여 사용하는 방법
    • 익명 함수: function에 함수명을 작성하지 않고 변수에 대입하는 방법
  • 매개변수 전달인자 : 자바처럼 엄격하지 않다. 선언이 안된 변수는 undefined로 자동설정된다.
