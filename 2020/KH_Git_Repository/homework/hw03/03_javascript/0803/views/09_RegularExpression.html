정규표현식

정규표현식의 객체 생성

$& , &` , &'



() 소괄호와 |

여러 문자열 중에 해당하는 것을 검색하려면 (str1 | str2 | str3) 식으로 사용가능



플래그 문자

  • g:_비교수행, 이 플래그가 없으면 _ 매칭만 처리
  • i: _고 비교하되 _만 처리



Anchor 문자

문자열의 앞^과 뒤$를 구분해주는 정규표현식 기호



수량 문자

a*: a가 0개이상
a+: a가 적어도 1개이상
a?: a가 0개또는 1개

{}를 이용하여 수량 표현

a{5}: a가 5개
a{2,5}: a가 2개~5개
a{2,}: a가 2개 이상
a{,2}: a가 2개 이하

메타문자와 [] 대괄호

각각의 메타문자는 특별한 의미의 문자열 양식을 나타내고
대괄호 묶음 안의 내용은 문자의 범위를 나타낸다.
대괄호 안의 ^은 시작이 아니라 특정 문자열 제외를 뜻한다.



추가 메타 문자

\d: 숫자
\w: 영어, 숫자, 언더바(_)
\s: 공백(tab, 띄어쓰기, 줄바꿈)
\D: 숫자빼고 다
\W: 알파벳, 숫자, 언더바 빼고 다
\S: 공백문자 빼고 다

Escaping

예약어를 실제 문자값으로 사용하려면, \(백슬래시)를 사용하여 escaping을 해야함.
