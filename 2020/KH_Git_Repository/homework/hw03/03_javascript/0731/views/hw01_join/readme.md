# python flask 웹서버 만들기

- 가상환경 만들기
  - 가상환경: 프로젝트별로 요구하는 환경 버젼이 달라져서 충돌을 막기위해서 만들어졌음.

  - 가상환경 설치 명령어 (windows OS)
  ```
  python -m venv myvenv(가상환경 이름)
  ```

  - 가상환경 설치 명령어 (mac OS)
  ```
  python3 - m venv myvenv(가상환경 이름)
  ```

- 가상환경을 만들면, 가상환경에서 사용하는 라이브러리를 다운로드받아야한다.

  - flask 라이브러리 설치
  ```
  pip install flask
  ```

  - pymongo 라이브러리 설치
  ```
  pip install pymongo
  ```
