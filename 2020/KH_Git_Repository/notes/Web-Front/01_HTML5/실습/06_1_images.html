이미지 관련태그

src 속성에 경로를 설정하여 이미지 파일을 불러온다.

img태그는 인라인 요소

자주 사용하는 속성들

alt 속성

화면 낭독기는 이미지를 읽을 수 없기 때문에 대체되는 텍스트가 필요하며 다르게는 사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없을 경우에 대체 텍스트의 용도로 사용가능하다.


width와 height속성

사진의 높이와 너비 지정 가능 고정 단위나 가변단위로 지정 가능

고정 크기 단위(default: pixel)

고정크기단위: 화면사이즈가 바뀌어도 크기가 변하지 않는다.


가변 크기 단위(use %)

가변크기단위: 화면사이즈가 바뀌어도 크기가 변한다.

이미지 구역을 2개로 나누어 각각 링크 설정
