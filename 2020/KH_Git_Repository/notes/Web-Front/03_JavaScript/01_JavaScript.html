자바스크립트 사용법

  1. 태그에 직접 간단한 소스코드를 작성해서 실행하는 방법
  2. script태그에 작성하여 실행하는 방법
  3. 별도의 js파일로 작성해서 파일을 불러와서 실행하는 방법

HTML 태그 접근

  • getElementById('아이디명')
    • 아이디는 겹치면 안되며, 유일해야한다.
    • 실은 중복해도 상관없지만, 아이디는 유일하게 사용하는게 좋다..

  • getElementsByName('이름')
    • name이 같은 태그가 여러개일 수 있다.
    • 여러개이기때문에 배열에 담아서 리턴이 된다.

  • getElementsByClass('클래스이름')
    • 클래스이름이 같은 태그가 여러개일 수 있다.
    • 여러개이기때문에 배열에 담아서 리턴이 된다.

  • getElementsByTagName('태그이름')
    • 태그이름이 같은 태그가 여러개일 수 있다.
    • 여러개이기때문에 배열에 담아서 리턴이 된다.
