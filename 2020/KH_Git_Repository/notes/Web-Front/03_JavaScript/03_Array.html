배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수 도 있고
크기를 정하고 선언할 수 있음.



배열 메소드

array도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드가 존재한다.

indexOf()

배열에서 요소가 위치한 인덱스를 리턴한다.


concat(배열명)

여러개의 배열을 결합한다.

join()

배열을 결합하여 문자열로 반환

reverse()

배열의 순서를 뒤집기

sort()

배열을 오름차순/내림차순 으로 정렬


push()와 pop()

push(): 배열의 끝에 요소를 넣는다
pop(): 배열의 끝에있는 요소를 뺀다.



shift( )와 unshift( )

shift(): 배열의 맨앞에 있는 요소를 제거
unshift(): 배열의 맨앞에 있는 요소 추가



slice()와 splice()

slice(): 배열에 있는 요소를 뽑아내기
splice(): 배열의 인덱스 위치 요소 제거 및 추가


toString()

배열을 문자열로 반환
join()메소드와 같은 기능을 하고 있지만
join()은 메소드 실행시 아무것도 넣지 않으면 ','로 합쳐지고
괄호안에 매개변수를 넣는다면, 그것이 구분자 역할을 하면서 그걸로 합쳐짐

반면, toString()은 매개변수안에 구분자가 없어서 기본적으로 합치는 기능만 함.
