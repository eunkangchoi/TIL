> # 배열

```javascript
function add(var var1, var var2){
  return var1+var2;
}

var 변수명 = [ 1, '123', true, add(1,2)]
```

<br>

- 배열 선언
  - new 연산자와 Array객체를 통한 배열 선언

```javascript
//크기를 정하지 않은 배열 선언
var 변수명 = new Array();

//크기를 정한 배열 선언
var 변수명= new Array(개수);
```

<br>

- 배열 초기화

```javascript
// new 연산자를 이용한 초기화
var 배열변수 = new Array(값1, 값2, 값3, ..., 값n);

//[]를 이용한 초기화
var 배열변수 = [값1, 값2, 값3, ... , 값n];

```
