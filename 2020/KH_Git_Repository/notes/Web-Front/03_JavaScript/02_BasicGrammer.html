변수 선언

함수 내부에서 'var 변수명;' 으로 선언하면 지역변수가 된다.

함수 내부에서 '변수명;' 으로 선언하거나
함수 밖에서 '변수명;' 혹은 'var 변수명;'으로 선언하면 전역변수가 됨.
전역변수와 지역변수가 동일한 이름인 경우 함수 내부에서 변수명을 호출하면 지역변수가 우선권을 가짐
전역변수 사용 시 'window.변수명' 혹은 'this.변수명' 으로 표현하여 지역변수와 구분
전역변수는 해당 window내에서 어디든 사용할 수 있으며, 지역변수는 해당 함수 내에서만 사용 가능.



자료형

자바스크립트에서는 자료형별로 변수 타입이 지정되지 않고
리터럴에 의해 자료형을 결정한다.

데이터 형변환

문자열과 숫자의 '+'연산

문자열 + 숫자 = 문자열
숫자 + 숫자 + 문자열 = 숫자가 먼저 계산되고 뒤의 문자열과 합쳐진다.

typeof() 연산자

값의 자료형을 확인하는 연산자
선언 시 자료형을 지정하지 않기 때문에 변수명만으로 데이터의 자료형 확인이 불가능하다.
