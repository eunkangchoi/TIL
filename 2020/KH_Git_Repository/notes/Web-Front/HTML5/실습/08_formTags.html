폼 관련 태그

form태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그
form태그 내의 input태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다. * action 속성: 폼에 입력된 값들을 전송받을 서버의 클래스명(페이지명) 입력
* method 속성: get/post 방식으로 전송방식을 지정한다.

submit버튼을 눌렀을 때 method에 지정한 방식으로 action에 명시한 경로에 input태그의 값을 전달한다.
