JQuery 개요

JQuery 장점

  1. DOM과 관련된 처리를 쉽게 구현할 수 있음.
  2. Ajax 통신/ 이벤트 처리등을 폭넓게 지원
  3. 별도의 플러그인을 통해 차트/슬라이드쇼/테이블을 간단히 구현 가능

JQuery 연결

  1. JQuery 홈페이지에서 js파일을 다운로드받아서 연결하는 방법
  2. CDN(Connect Delivery Network)을 통한 연결

$(document).ready()

페이지를 로드한 후 ready메소드를 실행한다.
자바스크립트에서는

window.onload();
JavaScript에서의 window.onload속성은 복수(여러) 개의 설정시 마지막에 쓴것으로 덮어지지만 JQuery는 여러개 설정하면 모두 연달아서 수행한다.
