이벤트 관련 속성

이벤트 핸들러의 매개인자로 event객체 전달
인라인에서는 event말고는 다른 키워드는 불가

이벤트 연결과 종류

on()과 off()

on(): 이벤트연결
off(): 연결된 이벤트 제거

기본상태
